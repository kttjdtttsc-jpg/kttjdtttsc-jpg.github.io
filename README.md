# Digital Footprinter

## About
A cybersecurity tool that reveals what your browser exposes about your device every time you visit a website. 

## How It Works  
When a user clicks the "Scan My Footprint" button, Javascript collects data from browser objects such as `navigator` and `screen`. The data is displayed on the page along with a Privacy Risk Score. The site also demonstrates canvas fingerprinting and WebGL to show some advanced fingerprinting methods a lot of sites also employ. 

## Features 
- Collects browser data including user agent, platform, language, and more
- Demonstrates canvas fingerprinting using the Canvas API 
- Demonstrates WebGL fingerprinting using the WebGL API
- Calculates a privacy risk score based on how much data is exposed
- Dark terminal inspired theme
