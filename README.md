# Browser Checker

## About
A cybersecurity tool that reveals what your browser exposes about your device every time you visit a website. 

## How It Works  
When a user clicks the "Scan My Footprint" button, Javascript collects data from browser objects such as `navigator` and `screen`. The data is displayed on the page along with a Privacy Risk Score that gives some recommendations on what to do if your score is above a certain threshhold. The site also demonstrates canvas fingerprinting and webgl to show some advanced fingerprinting methods alot of sites also employ. 

## Features 
- Collects browser data includer user agent, platform, language, and more
- Demonstrates canvas fingerprinting using the Canvas API 
- Demonstrates WebGL using 
- Calculates a privacy risk score based on how much data is exposed
- Dark terminal inspired theme
